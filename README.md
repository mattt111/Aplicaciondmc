# Aplicaciondmc

# Proyecto Módulo 1 - Fundamentos de Python

## 1. Descripción
Esta es una aplicación interactiva desarrollada en **Streamlit** que integra los conceptos fundamentales de programación aprendidos durante el Módulo 1 del curso[cite: 156]. La app permite gestionar actividades financieras, evaluar presupuestos y calcular retornos de inversión de forma dinámica.

## 2. Estructura del Proyecto
[cite_start]El proyecto consta de los siguientes archivos principales[cite: 157]:
* [cite_start]**app.py**: Archivo principal que contiene la lógica de Streamlit y los 4 ejercicios[cite: 158].
* [cite_start]**requirements.txt**: Archivo con las librerías necesarias para ejecutar la app (Streamlit)[cite: 159].
* [cite_start]**README.md**: Documentación del proyecto[cite: 159].
* **Logodedmc.png**: Logo institucional utilizado en la interfaz.

## 3. Contenido de la Aplicación
[cite_start]La aplicación se divide en 5 secciones accesibles desde el menú lateral[cite: 13, 19]:
* [cite_start]**Home**: Presentación del estudiante y objetivos del proyecto[cite: 23].
* [cite_start]**Ejercicio 1**: Verificador de presupuesto simple (Variables y Condicionales)[cite: 42].
* [cite_start]**Ejercicio 2**: Registro de actividades en listas y diccionarios[cite: 60].
* [cite_start]**Ejercicio 3**: Cálculo de retorno esperado usando `map` y `lambda`[cite: 92].
* [cite_start]**Ejercicio 4**: Modelado de actividades mediante Programación Orientada a Objetos (POO)[cite: 112].

## 4. Instrucciones de Ejecución
[cite_start]Para ejecutar este proyecto de forma local[cite: 160]:
1. Clonar el repositorio.
2. Instalar las dependencias:  
   [cite_start]`pip install -r requirements.txt` [cite: 161]
3. Ejecutar la aplicación:  
   [cite_start]`streamlit run app.py` [cite: 162]
