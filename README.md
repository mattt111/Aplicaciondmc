# Aplicaciondmc

# Proyecto Módulo 1 - Fundamentos de Python

## 1. Descripción
Esta es una aplicación interactiva desarrollada en **Streamlit** que integra los conceptos fundamentales de programación aprendidos durante el Módulo 1 del curso[cite: 156]. La app permite gestionar actividades financieras, evaluar presupuestos y calcular retornos de inversión de forma dinámica.

## 2. Estructura del Proyecto
El proyecto consta de los siguientes archivos principales[cite: 157]:
**app.py**: Archivo principal que contiene la lógica de Streamlit y los 4 ejercicios.
**requirements.txt**: Archivo con las librerías necesarias para ejecutar la app (Streamlit).
**README.md**: Documentación del proyecto.
**Logodedmc.png**: Logo institucional utilizado en la interfaz.

## 3. Contenido de la Aplicación
La aplicación se divide en 5 secciones accesibles desde el menú lateral:
**Home**: Presentación del estudiante y objetivos del proyecto.
**Ejercicio 1**: Verificador de presupuesto simple (Variables y Condicionales).
**Ejercicio 2**: Registro de actividades en listas y diccionarios.
**Ejercicio 3**: Cálculo de retorno esperado.
**Ejercicio 4**: Modelado de actividades mediante Programación Orientada a Objetos (POO)

## 4. Instrucciones de Ejecución
Para ejecutar este proyecto de forma local:
1. Clonar el repositorio.
2. Instalar las dependencias:  
   [cite_start]`pip install -r requirements.txt` 
3. Ejecutar la aplicación:  
   `streamlit run app.py`
